# Árbol Binario en Java

## ¿Qué es un árbol binario?

Un árbol es una estructura de datos jerárquica formada por nodos. Existe un nodo raíz y a partir de él se desprenden otros nodos llamados hijos.

Un árbol binario es un tipo de árbol en el que cada nodo puede tener máximo dos hijos: izquierdo y derecho.

## ¿Cómo se implementó?

Se creó una clase Nodo con:
- Un dato entero
- Referencias a hijo izquierdo y derecho

Luego se creó la clase ArbolBinario con métodos para:
- Insertar datos
- Recorrer en inorden
- Buscar un valor

Finalmente, se implementó un menú en consola para interactuar con el usuario.
